What is a System Curve?

The system curve represents the relationship between flow rate and required head in a piping system. It includes static head and friction losses.

As flow rate increases, friction losses increase proportionally to velocity squared, causing the system curve to rise.

How is the System Curve Calculated?

Total system head is calculated as:

Total Head = Static Head + Friction Loss

What are Pump Characteristic Curves?

Pump characteristic curves describe how a pump performs at different flow rates. The primary curve shows Head vs Flow Rate.

Each pump has unique curves depending on:

  • Impeller diameter
  • Rotational speed
  • Design geometry
  • Fluid properties

What Factors Affect Pump Curves?

  • Impeller trimming
  • Speed changes (Affinity Laws)
  • Fluid viscosity
  • Wear and mechanical condition

Changing speed shifts the curve according to pump affinity laws.

How is the Actual Operating Flow Rate Determined?

The actual operating point occurs at the intersection of:

  • System Curve
  • Pump Curve

At this intersection:

Pump Head = System Required Head

This determines the real discharge flow rate and operating head of the pump.
